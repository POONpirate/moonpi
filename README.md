# README #

MoonPi - A launcher for moonlight embedded.

### How do I get set up? ###

curl -s https://raw.githubusercontent.com/POONpirate/moonpi-apt/refs/heads/main/moonpi-bootstrap.sh | bash

### How do I uninstall? ###

sudo apt remove moonpi

On pi os 32bit full you can run this command to disable auto-switching off the desktop.
sudo touch /tmp/disable-vt-switch
